# Requirements

## Required Operating Systems
* CentOS 6.x or CentOS 7.x
* Mac OS X (experimental)
    
## Required compilers
* GCC 4.8.5 or higher
* CMake 3.3 or higher
    
## Required libraries
* libpcap-devel
* boost-devel
* boost-static

## Build steps
1. `mkdir build && cd build`
2. `cmake .. && make`
